# Calendario Compliance

Sistema de gestión de cumplimiento regulatorio para empresas con programas de Comercio Exterior (IMMEX, PROSEC, Padrón, IVA/IEPS).

## Tecnologías

- React + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Supabase (base de datos, autenticación, RLS)

## Desarrollo local

```sh
git clone <url-del-repo>
cd calendario-compliance
npm install
npm run dev
```

## Deploy

El proyecto se despliega en Vercel conectado al branch `main` del repositorio.
